# Remixicon x Sveltejs

A collection of icons made as independent SvelteJS components.

---
### Installation
```html
npm i remixicon-sveltejs
```
---
### Usage
```html
<script>
  import { AddIcon } from 'remixicon-sveltejs';
</script>

<AddIcon />
```
